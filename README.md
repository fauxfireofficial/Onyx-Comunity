# Pixo App - Onyx Community

This is the core source code for the Pixo App project, customized for Onyx Community.

## Structure
- `www/`: Frontend web assets (HTML, CSS, JS).
- `server/`: Backend scripts (push notification logic).
- `assets/`: App icons and splash screens.

## Getting Started
To get this project running:
1.  Clone the repo.
2.  Install dependencies: `npm install`.
3.  For notifications, ensure `serviceAccountKey.json` is configured in the `server/` directory (excluded for security).
4.  Run locally or build for Android/iOS using Capacitor.
